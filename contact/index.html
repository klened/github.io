문의 시 도움이 되는 정보

빠르고 정확한 답변을 위해 다음 정보를 함께 알려주시면 좋습니다:

  • 문의 대상 페이지 URL (예: https://klened.github.io/youth/)
  • 발생한 문제의 구체적인 상황 (스크린샷이 있다면 더 좋습니다)
  • 사용 중인 기기 및 브라우저 (예: 아이폰 Safari, PC Chrome 등)
  • 잘못된 정보 제보의 경우 — 정확한 출처(공식 사이트 링크) 첨부

응답 시간 안내

본 사이트는 1인 운영으로 즉시 응답이 어려울 수 있습니다. GitHub 이슈는 영업일 기준 2~5일 이내에 확인 및 답변드리는 것을 목표로 합니다. 단순 정보 수정 제보의 경우 별도 답변 없이 빠르게 반영되는 경우도 있으니 양해 부탁드립니다.
