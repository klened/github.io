이런 내용으로 문의해주세요

  • 잘못된 정보 제보: 대출 한도/금리, 지자체 출산장려금 등 사실과 다른 내용을 발견하셨다면 알려주세요.
  • 추가 정보 요청: "이런 정책도 비교해줬으면 좋겠어요", "○○구 지원금이 누락됐어요" 등.
  • 기능 제안: 새로운 도구·계산기·가이드에 대한 아이디어.
  • 오류 신고: 사이트가 동작하지 않거나 표시가 이상한 경우.
  • 광고 관련: 광고가 부적절하다고 판단되는 경우 신고해주세요.
  • 제휴 / 협력 문의: 콘텐츠 협력, 데이터 제휴 등.

문의 시 도움이 되는 정보

빠르고 정확한 답변을 위해 다음 정보를 함께 알려주시면 좋습니다:

  • 문의 대상 페이지 URL (예: https://klened.github.io/loan/)
  • 발생한 문제의 구체적인 상황 (스크린샷이 있다면 더 좋습니다)
  • 사용 중인 기기 및 브라우저 (예: 아이폰 Safari, PC Chrome 등)
  • 잘못된 정보 제보의 경우 — 정확한 출처(공식 사이트 링크) 첨부

답변이 어려운 문의

다음과 같은 문의는 본 사이트에서 답변드리기 어렵습니다. 해당 기관 또는 전문가에게 문의해주세요:

  • 개별 대출 승인·심사 결과 관련 문의: 해당 금융기관 또는 주택도시기금 콜센터(1599-0001)
  • 지자체 출산장려금 신청 관련 문의: 거주지 시·군·구청 또는 주민센터
  • 육아휴직급여·출산휴가 관련 문의: 고용노동부 고객상담센터(1350)
  • 법률·세무 자문: 변호사·세무사 등 전문가

응답 시간 안내

본 사이트는 1인 운영으로 즉시 응답이 어려울 수 있습니다. 영업일 기준 2~5일 이내에 답변드리는 것을 목표로 합니다. 단순 정보 수정 제보의 경우 답변 없이 빠르게 반영되는 경우도 있으니 양해 부탁드립니다.
